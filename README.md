# io-spin

Little terminal spinner lib for Node.js and io.js

## Installation

```bash
$ npm install io-spin
```

## Example

```javascript
spin.start('Downloading:', 'Spin1')
spin.restart('Computing:', 'Box1')
spin.stop()
```
